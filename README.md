# Brend Çanta Qalereyası

Bu layihə premium brend çantaların təqdim olunduğu veb saytın hazırlanmasını əhatə edir.

## Layihə haqqında
Brend Çanta Qalereyası yalnız orijinal Gucci, Chanel, Prada və Hermès çantalarının satışını həyata keçirən butik konseptidir.

## Məqsəd
Müştərilərə eksklüziv və yüksək keyfiyyətli alış təcrübəsi təqdim etmək.

- Məhsul siyahısı
